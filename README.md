## Deployment

This site is deployed via GitHub Pages:

1. Files are located in the repository root
2. GitHub Pages serves `index.html` automatically
3. Site is live at: `https://[username].github.io/fratelli/`

## Local Development

Open `index.html` in a web browser to view locally.

## Structure

- `index.html` - Main page with contact information and booking link
- `styles.css` - Responsive stylesheet with mobile-first design
- `.gitignore` - Excludes system files from version control
